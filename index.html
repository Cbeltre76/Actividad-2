2- ¿Cómo se gestionan los procesos y cuál es la importancia de la multitarea en la eficiencia del sistema?

La multitarea o multitasking en dispositivos electrónicos se refiere a la capacidad de ejecutar varias aplicaciones o procesos al mismo tiempo. En otras palabras, es una característica permite a los usuarios cambiar rápidamente entre diferentes tareas sin tener que cerrar una aplicación antes de abrir otra.

3- ¿Qué medidas de seguridad se pueden implementar a nivel de sistema operativo para proteger una red, especialmente en entornos empresariales?

1. Actualizaciones regulares del sistema operativo: Mantén el sistema operativo actualizado con los últimos parches de seguridad. Las actualizaciones suelen solucionar vulnerabilidades conocidas que podrían ser explotadas por los atacantes. 2. Firewall: Utiliza un firewall para controlar el tráfico de red entrante y saliente. Esto puede ayudar a bloquear el tráfico no deseado y proteger el sistema contra ataques. 3. Antivirus/Antimalware: Instala y actualiza regularmente software antivirus o antimalware para detectar y eliminar posibles amenazas. 4. Políticas de seguridad: Implementa políticas de seguridad que especifiquen cómo deben configurarse los sistemas operativos, como contraseñas seguras, bloqueo automático de sesiones inactivas, etc. 5. Control de acceso: Utiliza mecanismos de control de acceso para restringir quién puede acceder al sistema y qué recursos pueden utilizar. 6. Auditoría de seguridad: Realiza auditorías de seguridad regularmente para identificar posibles problemas de seguridad y corregirlos. 7. Encriptación de datos: Utiliza la encriptación para proteger los datos sensibles almacenados en el sistema operativo y en la red. 8. Detección de intrusiones: Implementa sistemas de detección de intrusiones para identificar y responder a posibles intentos de acceso no autorizado. 9. Respaldos regulares: Realiza respaldos regulares de los datos importantes y asegúrate de que estos respaldos estén almacenados de forma segura, de modo que puedan ser restaurados en caso de un ataque o fallo del sistema. 10. Restricciones de privilegios: Limita los privilegios de usuario para reducir el riesgo de que se produzcan actividades maliciosas. 11. Monitoreo de actividad: Implementa herramientas de monitoreo de actividad para supervisar el tráfico de red y el comportamiento del sistema en busca de actividades sospechosas.
